## Quick Start

1. **Installation**
   ```sh
   uv sync
   ```

2. **Basic Usage**
   ```sh
   pypl2mp3 import <playlist_url>
   pypl2mp3 play
   ```

3. **Getting Help**
   ```sh
   pypl2mp3 --help
   pypl2mp3 <command> --help
   ```
